# 🏎️ Проект: Автономный робот для Ралли

Добро пожаловать в репозиторий команды по разработке гоночного робота! Проект создается в рамках весеннего интенсива (30 марта – 3 апреля 2026 г.) для подготовки к соревнованиям «Ралли по коридору» (Регламент РобоФинист 6.1).

---

## 🛠 Технические требования (Check-list)
Чтобы пройти техническую комиссию, наш робот должен строго соответствовать следующим параметрам:

* **Автономность:** Полностью самостоятельное движение. [cite_start]Любое внешнее или телеуправление запрещено[cite: 24].
* [cite_start]**Конструкция:** Четырехколесный автомобиль с передним, задним или полным приводом[cite: 28]. 
* [cite_start]**Рулевое управление:** Поворот осуществляется исключительно за счет управляемых передних колес вокруг вертикальной оси (схема Аккермана)[cite: 28, 30].
* [cite_start]**Габариты:** Робот должен быть меньше **500 мм** в длину, **400 мм** в ширину и **300 мм** в высоту[cite: 16, 18, 20].
* [cite_start]**Безопасность:** На передней части обязательно наличие мягкого бампера толщиной не менее **10 мм** (поролон, пенополиуретан, пористая резина)[cite: 31].
* [cite_start]**Колеса:** Шины не должны удерживать лист бумаги А4 плотностью 80 г/м² дольше 2 секунд[cite: 27].

## 🧩 Электроника и компоненты
* **Контроллер:** Рекомендуется использование плат Arduino, ESP32 или Raspberry Pi Pico.
* **Датчики:**
    * [cite_start]Ультразвуковые или ИК-дальномеры для удержания дистанции от бортов (высота бортов — не менее 150 мм)[cite: 50].
    * [cite_start]Датчики линии для распознавания черных меток перед препятствиями[cite: 53, 55].
* [cite_start]**Привод:** Тяговый мотор на ведущую ось и сервопривод для управления углом поворота колес[cite: 28].

---

## 🚀 Алгоритм и препятствия
Для успешного прохождения трассы робот должен уметь:

1. [cite_start]**Остановка на «Стоп-линии»:** При обнаружении черной полосы шириной 250 мм робот обязан совершить полную остановку, не выходя проекцией за линию, и только затем продолжить движение[cite: 62, 63, 64].
2. [cite_start]**Преодоление «Камней»:** Участок с неровностями высотой до 35 мм[cite: 45, 72].
3. [cite_start]**Проезд «Эстакады»:** Двускатная горка с углом подъема и спуска не более 20°[cite: 79].
4. [cite_start]**Реакция на знаки:** Перед «Камнями» и «Стоп-линией» нанесены черные линии (знаки оповещения) на расстоянии 1 метра, которые нужно использовать для смены алгоритма[cite: 65, 73].
