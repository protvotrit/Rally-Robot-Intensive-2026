# 🏎️ Проект: Автономный робот для Ралли (Интенсив 2026)

Репозиторий команды по разработке гоночного робота для соревнований «Ралли по коридору». 
[cite_start]Проект выполнен в рамках весеннего интенсива (30 марта - 3 апреля)[cite: 4].

## 🛠 Технические характеристики (Регламент 6.1)
* [cite_start]**Тип управления:** Полностью автономный. [cite_end]
* [cite_start]**Шасси:** Четырехколесное с рулевым управлением Аккермана[cite: 28, 30].
* [cite_start]**Габариты:** < 500х400х300 мм[cite: 16, 18, 20].
* [cite_start]**Обязательный элемент:** Мягкий передний бампер (толщина >= 10 мм)[cite: 31].

## 🧩 Компоненты
* [cite_start]**Контроллер:** Arduino / ESP32 / Raspberry Pi Pico.
* [cite_start]**Датчики:** Ультразвуковые/ИК датчики расстояния для ориентации вдоль бортов[cite: 34].
* [cite_start]**Привод:** Мотор-редуктор на заднюю/переднюю ось и сервопривод для поворота колес[cite: 28].

## 🚀 Как это работает
1. [cite_start]**Движение:** Робот ориентируется по бортам высотой не менее 150 мм[cite: 50].
2. [cite_start]**Препятствия:** Алгоритм распознает черные линии (знаки оповещения) перед «Камнями» и «Стоп-линией»[cite: 53, 65, 73].
3. [cite_start]**Остановка:** На препятствии «Стоп-линия» (черная полоса 250 мм) робот совершает полную остановку[cite: 62, 63].

## 📁 Инструкция по сборке
1. Скачайте STL-модели из папки `/3D_Models`.
2. Распечатайте детали (корпус, рулевая рейка, колеса).
3. Соберите электронику по схеме в папке `/Docs`.
4. Загрузите код из `/Code`.
